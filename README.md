### Uses: 
> 1. State react-tracked + useReducer + immer
> 2. Styles: emotion
> 3. Animation: framer motion
> 4. Notify: toastify 
> 5. Firebase DB

### Start
```bash
# install dependencies 
make install-yarn

# starting JSON server http://localhost:4001/posts + http://localhost:4000
make start 
```
